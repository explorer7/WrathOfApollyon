Low-Poly Terrain
Version 1.0.0
11/19/2015

To get started, select a sample terrain configuration file in LowPolyTerrain/Samples, or create a new one by selecting Assets->Create->LowPolyTerrain Config from the menu bar.

In the inspector window, adjust terrain parameters and click "build" to generate the terrain in the current scene.

Visit http://lowpolyterrain.weebly.com for documentation.
